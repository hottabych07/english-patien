# The English Patient

Backend - Django rest framework
Frontend - React+redux

## Установка
**1.**
```
docker-compose build
```

**2.**
```
docker-compose up
```

## Разработка

**Запуск локального стенда:**
```
docker-compose up
```

**Остановка локального стенда:**
```
docker-compose stop
```
