# The English Patient

Backend - Django rest framework, Frontend - React+redux

[Screenshots](https://imgur.com/a/GR4Zl2p)

## Install
**1.**
```
docker-compose build
```

**2.**
```
docker-compose up
```

## Developer

**Start local stand:**
```
docker-compose up
```

**Stop local stand:**
```
docker-compose stop
```
